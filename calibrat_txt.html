Online Tool

Calibration Report (CALIBRAT.TXT)

View the calibration results, that is, the information saved in the file, CALIBRAT.TXT, by the AUTO_CALIBRATE.gcode (or the CALIBRATE_25MM.gcode) command during auto-calibration.
Select the calibration results file (i.e. CALIBRAT.TXT)
Add a note (optional, included in the report)

This tool scans through the file (e.g. CALIBRAT.TXT) to extract basic information such as the firmware version number and the printer's configuration and geometry. If the file contains the output of an auto-calibration command (G33), these results are listed as well.

The tool also extracts and displays probe results and the bed level mesh. If it finds both, it will use the same bilinear z-offset correction as the printer to assess the overall "quality" of the machine's calibration.
